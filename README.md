# TMG Consultation TMG

Site web : TMG Consultation

# 🚀 Objectifs du projet

- Présenter les services et réalisations de TMG Consultation.
- Mettre en valeur les projets et références clients.
- Offrir une expérience utilisateur moderne avec des animations avancées.
- Permettre à l'équipe de gérer facilement le contenu via une interface d'administration.
- Optimiser les performances, le référencement (SEO) et l'accessibilité.

# 🏗️ Architecture Technique

## Frontend

### Next.js 14 (App Router)

- Server-Side Rendering (SSR)
- Static Site Generation (SSG)
- Excellentes performances
- SEO optimisé
- Routing moderne et simplifié

### Tailwind CSS

- Développement rapide et maintenable
- Design responsive
- Système de design flexible

### GSAP + ScrollTrigger

- Animations avancées
- Effets déclenchés au défilement
- Expérience immersive inspirée des agences créatives modernes
